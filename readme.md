# Handwritten Digit Recognition using CNN

A Deep Learning project that recognizes handwritten digits (0–9) using Convolutional Neural Networks (CNN) and the MNIST dataset.

---

# Features

- Handwritten digit prediction
- CNN-based Deep Learning model
- Flask web application
- Image upload support
- Real-time prediction
- Simple and clean UI

---

# Tech Stack

- Python
- TensorFlow / Keras
- Flask
- NumPy
- Pillow
- HTML

---

# Dataset

This project uses the MNIST dataset containing:
- 60,000 training images
- 10,000 testing images
- 28x28 grayscale handwritten digits
